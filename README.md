唯心主义来讲，数据是人类一切活动的记录和产物，唯物主义来讲，数据是宇宙万物的存在和运动记载。透过人类历史来观察人是怎样认识和改造这个世界的，翻阅不同阶段各种哲学和科学，总体概括起来就是意识对于所接触数据做出处理的认知，靠脑意识操纵的肢体来改造抑或靠肢体产生的工具来方便点的改造，从史前文明石器时代农耕文明蒸汽时代和电气时代再到目前的信息时代，无疑是对数据的认知和处理的改善，加强了人类认识和改造世界的工具，特别是现有的信息时代计算机更多的扮演的角色还是辅助工具。预测下一个时代为什么会有突破性？是机器意识的不断改进升级，简单的单片机其内部电子信号的流动到一定规模的时候就会可以处理最简单的数据，再具集再聚集，其可以通过电信号控制数据命令操纵运动性工具，硅基等其他基机器的不懈改进和升级，最后将不仅仅是辅助我们改造世界，甚至可以作为整合的类人的主体行为的去改造世界。

### 数据定义和数据主义哲学

只有两种形式的数据，一种是描述性的，一种是运动性的，一般来讲，前种是静态的，后种是动态的，数据科学体系的发展完善，前一种是要保证数据存在的唯一真实性，对于第二种则是要努力探求其运动规律。

#### 数据时代起源

数据时代俨然不是一个新名词，其起源简而言之是从人类文明的起源语言和数字这些基础的数据工具开始，经历漫长的探索期，直到电气时代电的发现开始文明进程加速，由特殊的电器件半导体特殊导电性和图灵伟大的01构想的有机结合，构成了01搭配的数据帝国的基础（也可以说通信帝国，但通信多了就叫数据帝国吧）利用我们人类的才智赋给了01组合无限的意义，计算机应运而生，引用计算机基础知识，开始是0101这样数据处理为核心，向后散发了无限功能，利用0和1及其各种组合形成的数据，在机器端建构了另一个世界，多个计算机设备互联就是互联网，之间数据传输加速，因为人和计算机交互，所以也加速了人人间数据交互，文明加速，此部分描述详见《文明之光》对于人类文明进步体系中对于数据的详细认知。

那么此时，或者未来人类改造世界依靠什么呢，会是在人机系统更为成熟的情况下利用数据合理操控下的机器，机器有了近乎于人一样的功能，社会生产使得关于人的剥削成为了历史，“剥削，改进”机器才是未来的图景，而可见未来下人类的社会结构？必将是科学数据协议下的自由人联合体，既不是美丽的新世界也不是老大哥，倘若如此人类的社会系统必然还是没有稳定，一个稳定状态一定是联合体，如同蚁群一样，换一个角度，就像是目前的分布式服务器和区块链协议之于物，相应的分布式的群体和社会协议之于人。不得不承认目前的世界就是“美丽的新世界”这样的不公平和不均衡，充斥着各种各样的矛盾，仍旧是数据流动性障碍下少数对多数的控制，大多数个体仍旧缺乏全面发展的条件，但并不是所有矛盾都是靠维稳和内部调和能够解决的，只有共产主义社会这样对于数据运动的合理框架才会避免剥削，战争和流血，避免人类改造自然，探索自我的历史道路上的更多波折。

#### 数据运作方式

一切事物和人的运动和相互作用都是有关数据的交互流动。

物质通过数据决定意识，同理意识也通过数据作用于物质，抓住了数据，就是抓住了意识精神世界和物质现实世界沟通的桥梁，也就是发现了最接近马克思主义的最终科学，这样的科学似乎让我们发现几乎一切物质和意识运动的规律，进而重新定义一切规则准则，更好的走向共产主义社会，获得人的全面自由发展与解放。

至今为止，一切人类社会和文明发展的矛盾都可以归结为数据的矛盾，即意识对现实世界反应在数据上的不真实不准确性，数据流通中的不平衡不对称或者偏差，那么我们就看见了关于未来社会的规则制度的构想，人民遵循相应其发展的基于对数据作用规律的准则进行挖掘。

为什么是数据的运作方式？科研的观测莫不是一种观察自然，人文等等各种类型数据变动再用数学文字等人可理解的信息进行描述推断？哀吾生之须臾，人生之短暂渺小使得个体一切的活动都可以被抽象为数据和其操纵的事物上，只不过这个建模目前是庞杂难以完全实现的，但截至目前的计算机帝国的数据运作还是可见一隅的，结合相关学科的知识，由硬件层的数据采集获取到数据层构建的传输协议，底层开发协议建成了基础操作系统和数据库系统，在这个基础上进行的语言开发和软件开发的结果构成了大多数人所接触的数据运作端口，并在这上面进行对于虚拟世界数据的操作，以往的数据库是个人可以通过方法进行删改的，也就是说当今的虚拟世界的数据存在不稳定性和以人为意识转移的性质，但以区块链技术为核心的底层数据协议避免了人个体或者部分集体对于虚拟世界数据的非法操作，即在虚拟世界实现了唯一不可删改，也就是说除非服务器物理存在全部抹去，否则全体都能实现对于根源数据的监督把控，这对于一个人机交互，虚拟世界和现实世界融合愈加紧密的当下是不可小觑的革新，现实世界中的人和事物太容易为个人或集体的意志所左右了，显而易见并且常常发生的关于人的有资本家对于人的经济控制，上级对下级的政治控制，关于事的有中心化权力节点对数据的篡改，比如各种证书奖项造假，新浪微博的删改等，而这种不合理性往往难以矫正，原始数据难以得到留存，倘如事件对应的虚拟世界唯一可信，那么极大程度上会避免个或集体的意志的不合理运动。现实问题也就是说——只有保证数据的源头不被篡改，在虚拟世界实现更好的分配制度，社会主义建设才能避免发展中的不均衡不充分。

#### 数据科学与教育

数据科学框架下的教育，科学指引人的全面发展。

当前数据科学相关体系发展情况：

数据计算：包括人工智能机器学习神经网络这些学习类的算法优化改进，mapreduce此类计算框架；数据储存：包括区块链，分布式Hadoop等数据库技术和对应底层硬件的革新；数据传输：包括5G，云系统方案；数据硬件：量子计算，生物计算，纳米计算；

2009年某篇论文提到过：“一个建立全国集中统一数据中心的构想”就阐述了论基于学习型社会创建中的公共电子图书馆的发展，我们的学习资料我们的教育是不应该市场化的，教育和人的发展不能是靠资本牵着走，当然更不能是靠权力牵着走，是要符合人类发展和个体的潜能来定的，就应该发挥社会主义优势集中办数据产业，首当其中的就是打破目前的教育市场化，将一切的电子的学习资源公有开源。

尽管是在一个环境中，不同人对于现实数据的处理和建模，时间长了之后所产生的差异也是巨大的，一方面是认知的原因，另一方面则是环境，举个简单的例子：即便是同样入学大学的两个人，经过一段时间的大学生活和学习其看到的接触到的，体现在电脑上（虚拟世界）的差异是巨大的，一方面一个人的电脑可能是全是思想记录，技术学习，积极向上的，而另一个人却几乎全是杂七杂八的游戏和碎片，这其中差异除了意识本身的数据主动作用，我想还是环境的数据引导出了问题。我们的环境-不同地区空间的数据差异和作用形式差别太大了，引用现在信息系统研究的发现，开放式数据协议相对比市场式的认知评估，其体系运作将会更加快捷和客观。

数据科学重写文科教育

#### 打破数据流动性障碍

为什么要打破流动性障碍?矛盾既是一种数据的流动性障碍。

经济活动本质上就是一种钱的流动，钱太可以量化数据化，因而所以经济危机本身就是一种数据流动性障碍，当社会出现因为收入分配不合理以及资本剥削套利的情况导致两极分化矛盾突出的时候，必然会有部分群体想消费却消费不起，没有消费就没有生产，就会在生产者出现断档造成生产端失业的经济危机，政府的调控往往是阶段性的不持久不长效，其一般体现为周期性的经济危机（结构性和其他经济危机是同时夹杂着信息的不对称）。

社会矛盾，或者社会流动性障碍也是数据流动性障碍的一种，成因是不合理的政治架构导致了权利的交互板结，其一般体现为现有利益集团的流动性障碍和阶层固化等矛盾，新生代的优秀人群受之前利益集团的裙带关系打压迟迟不能完成社会阶层上升，造成政治经济资源的流动性障碍，不思进取没有价值造成改革障碍的原有利益集团为维护其地位而新锐却不能实现其个人价值的最大化，一般提现为阶级复制和阶级斗争。

信息不对称，或者说信息的流动性障碍涉及根本，影响一个人因为它影响一个人的三观以及根本性认知以及未来的个人发展，影响转换为知识的方式和内容并因此作用在在个体的言行举止为人处世生活学习工作方方面面，信息流动性障碍一般提现为方方面面的信息不透明不公开，比如存在的互联网长城，教育系统市场化同时令人堪忧，由钱约束的教育信息资源，政府的各种数据失真，信息监管错位，学校的交流交换评奖评优信息，互联网公司诸如百度对于搜索结果的肆意篡改，小道消息等等等等，由于互联网公司垄断或者政府信息化的不健全造成的消息的流动性障碍导致很多人不能及时提取到有效信息和知识，耽搁了其进一步发展，同时也造成了社会不公平，催生了各种咨询和中介公司。

数据的流动性障碍是上述矛盾之根本，一切皆是由于物理空间及其一切数据衍生物的不均衡不对称流动的障碍，前面已经说明对于数据引导打破信息知识的不对称促进人的全面发展，接下来将进一步说明利用数字经济，数字权力打破个人经济政治的不对称。

#### 数字经济

我们要创造价值，不要创造钱，钱永远不等同于价值，未来的数字货币这一串数据也永远不等同于价值，劳动工作服务科研，这些促进他人和社会进步的活动才会产生价值，简单说：农民实实在在供养人民，工人实实在在建造房屋，工程师科学家教师等实实在在推动社会进步，卓越管理者的组织协调这些才是价值的产生，只有这些基本法律框架内的行为，合理的价值产生才能拥有数字经济的交易权，明白这一点再去进行行之有效的建设中来

在人类社会的发展过程中，社会经济运行采取的基本形态大致有三种：

1.自己生产自己需要的东西，自给自足，这种经济运行形态称为自然经济。

2.用自己所生产的劳动产品同别人所生产的、不同的劳动产品相交换，从而取得自己所需要的东西，这种经济运行形态称为商品经济。

3.虽然不是自给自足，但取得自己需要的东西不是通过交换，而是通过社会中心机构集中的、统一的分配来取得，这种经济运行形态称为产品经济。

自然经济是与极其落后的社会生产力水平相适应的经济形态。商品经济是以社会化大生产为生产力基础的，是人类社会发展至今最基本的一种经济形态。商品经济经历了不同的发展阶段，市场经济是商品经济的发达阶段。产品经济是以生产力高度发达为基础的，并以全社会经济利益的一致性为前提，它是马克思对未来的共产主义社会的设想。

引用华罗庚计划经济的最大效用理论，1985年华罗庚在加州理工曾经对于资本主义经济和祖国的社会主义经济做过比较分析和基础的数学推导，讨论了计划经济的最优。当然还成功的预见对于数字经济和对应的未来产品经济的优势是：好在生产上的宏观不会超过地球, 微观不会小于一件商品，这样的产品经济便可以在未来巨大算力支持下，新技术确定的数据真实性的前提下成功实施。

劣币驱逐良币这个通俗易懂的经济学现象等等都会随之消失，让实实在在做好事，做有意义有价值的良币得以流通，让无意义无价值，走捷径，投机取巧的劣币得以断绝，诸多违背良心，不合理但是低成本的经济活动便会丧失其原有的可行性，数字经济让经济流动高质量且高效。

这个故事有个时代背景：当时将校们很少有不冒功的号称斩首多少多少，其多有假冒。追究起来，他们砍下来的很可能是当地老百姓的脑袋，所谓滥杀无辜。如果没人叫真，这些脑袋就是战功，大家升官发财，万事大吉。如果有人叫真，这些脑袋就可能成为罪证，这帮将校罪过不小。所以，将校的命运确实在相当大的程度上掌握在那位小吏的手里，尽管他的官未必及得上人家手下的一个排长。

张居正总结说：人们怕那些吏，一定要贿赂那些吏，并不是指望从他们手里捞点好处，而是怕他们祸害自己。合法地祸害别人的能力，乃是官吏们的看家本领。这是一门真正的艺术，种种资源和财富正要据此分肥并重新调整。

另一个是数据权力的流动性障碍的例子：李东阳说了：“老百姓的情况，郡县不够了解。郡县的情况，朝廷不够了解。朝廷的情况，皇帝也不够了解。”这大意是不错的。不过，按照他的说法，老百姓和皇帝之间只隔了两道信息关卡，即郡县和朝廷。实际上，在充分展开的情况下，老百姓和皇帝之间隔着七道信息关卡。直接接触老百姓的是衙役，这是第一关。衙役要向书吏汇报，这是第二关。书吏再向州县官员汇报，这是第三关。州县官员向府一级的官员汇报，这是第四关。府级向省级官员汇报，这是第五关。各省向中央各部汇报，这是第六关。中央各部向内阁（皇上的秘书班子）汇报，这是第七关。信息到达终点站皇上面前的时候，已经是第八站了。这还没有算府、省、中央各部的科、处、局和秘书们。即使在最理想的状态下，也不能指望信息经过这许多层的传递仍不失真。

更何况，信息在经过各道关卡的时候，必定要经过加工。在无数信息之中，注意了什么，没注意什么，选择什么，忽略什么，说多说少，说真说假，强调哪些方面，隐瞒哪些方面，什么是主流，什么是支流，说得清楚，说不清楚，这都是各级官吏每天面对的选择。
